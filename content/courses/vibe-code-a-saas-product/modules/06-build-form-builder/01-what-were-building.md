# What We're Building

Our third and most ambitious tutorial project: a form builder with drag-and-drop, style customization, and a real database.

## The App

A form builder where you can:

- Drag and drop field types (text input, textarea, select, checkbox) onto a canvas
- Reorder fields by dragging them
- Customize basic styles — labels, placeholder text, required/optional
- Preview the form as a respondent would see it
- Share a public URL where people can fill out the form
- View submissions in a simple dashboard

## Why This Project?

This is the project that bridges the gap between "I built a tutorial app" and "I can build my own SaaS."

A form builder touches almost every concept your SaaS will need:

- **User-facing interface** with interactive elements (drag and drop)
- **Database operations** — creating forms, storing submissions, querying data
- **Public vs. private pages** — the builder is private, the form URL is public
- **Data modeling** — how do you represent a dynamic form in a database?
- **Server actions** — saving forms, processing submissions

This is also where we introduce Turso and Drizzle — the database tools you'll use in your own SaaS.

## The Dubsado Connection

Forms were one of the first features we built into Dubsado. Our users — photographers, designers, coaches — needed to send questionnaires to their clients. The first version of our form builder was embarrassingly basic. But it worked. And users loved it because it solved a real problem.

That's the lesson I keep coming back to: you don't need the fanciest version. You need the version that works. We're not building Typeform here. We're building something functional that teaches you the patterns.

## What You'll Learn

By the end of this module, you'll understand:

- How drag-and-drop works in React
- How to set up and connect a database
- How to design a data model for dynamic content
- How to write **user stories** (our core concept for this module)
- Why your data strategy matters more than your UI
